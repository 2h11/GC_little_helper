GC little helper

You can select on your own, which of these features you want to use. All Features are configurable with an easy-to-use menu. Wishes, Feature- and Bookmark-Requests, Errors and other Support at: geoclub.de


German Version: http://www.amshove.net




Features

Global

  • Create your own Linklist for the Navigation
    • Like the original Navigation or as an additional Drop-Down List
    • Remove original gc.com links from navigatio
    • Configure which link in which order, name, ...
    • Select your links out of a list with all significant links on gc.com and other external links
    • Also there are links with special features: Nearest-List, Nearest-Map, Nearest-List without Founds, User-Search as Popup
    • Create own custom links
  • Searchfield in navigation
  • Change the width of the whole gc.com page
  • Inserts your home coordinates in the searchfield of the start-page
  • Hide Feedback-Button
  • Hide Disclaimer
  • Adds a link from google maps to geocaching.com
  • Configuration-page to (de-)activate and configure all features

Nearest List

  • Automatic redirect from Nearest List to Map
  • Log It-Link in the Nearest List
  • Change background of found Caches to green instead of grey

Profile

  • Your own Linklist in the Sidebar
  • Collapsable boxes in the Sidebar
  • Edit-Link beside own Caches

Maps

  • Linkliste on Beta-Map (choose which links to show on map)
  • Display homezone on Map
  • Change the width of the "old" Map
  • Set "old" Map as default

Listings

  • Automatic show of all Logs (configure threshold)
  • Mail-Link beside Username (Cache- and TB-Listings) - also adds Cache-/TB-Name to mail
  • Hide empty Cache Notes (show via Link)
  • Hide Cache Notes completely
  • Decrypt hint automatically
  • Link to google maps
  • Automatically activates the dynamic Map
  • Log cache directly in listing
  • Shows the namen of the Coin-Series in TB-Listing
  • gc.com Bug fixed: Linebreaks in decryptet hints

Logging

  • Default-Log-Type for caches and TBs
  • Cache- & TB-Log-Signature
  • Use #found# for the amount of caches found +1 in the Log-Signature
  • Log the caches directly in listing
  • Shortcut: F2 to submit a log
  • Smilies and BBCode beside log-field
  • Transfer Tracking-Nummer to TB-Log-Field

Mail-Form

  • Mail-Signature
  • Save state of checkboxes

Friendlist

  • Additional links
    • Amount of caches as link to found-list
    • Link to gallery
    • Link to hidden caches
  • Amount of caches found since last access

Bookmarks

  • "Show in google maps"-Link
